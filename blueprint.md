# Superbuilders Project Blueprint

## Overview
**Superbuilders** is a landing page dedicated to solopreneurs and one-person developers.
**Core Value:** "Building massive value, solo."
**Goal:** Drive newsletter subscriptions and sell educational products (Courses, E-books).

## Architecture & Tech Stack
- **Framework:** None (Vanilla HTML, CSS, JS)
- **Styling:** Modern CSS (CSS Variables, Flexbox/Grid, OKLCH colors)
- **Components:** Web Components (Custom Elements)
- **Theming:** Full Dark/Light mode support with system preference detection.

## Features
1.  **Hero Section:** Strong value prop + Newsletter Subscription Form.
2.  **Weekly Issues:** Archive of past newsletter issues.
3.  **Academy:** Section for Online Courses.
4.  **Bookshelf:** Section for E-books.
5.  **Theme Toggle:** User-controlled Light/Dark mode switch.
